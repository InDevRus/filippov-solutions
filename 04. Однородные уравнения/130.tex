Для данных кривых будем находить дифференциальное уравнение, затем при выраженном $y'\br{x} = f\br{x; y\br{x}}$ составим уравнение изогональных траекторий по правилу
$\zed'\br{x} = \dfrac {f\br{x; \zed\br{x}} - 1} {1 + f\br{x; \zed\br{x}}}$ (см. задачи \textbf{40}, \textbf{46} и \textbf{48}). Полученные уравнения будут однородными.

\begin{enumerate}
	\item $y = x \ln\br{Cx}$.
	$C = \dfrac {\exp\br{\frac {y} {x}}} {x}$.
	$0 = \dfrac {\frac {y'x - y} {x^2} \cdot \exp\br{\frac {y} {x}} \cdot x - \exp\br{\frac {y} {x}}} {x^2}$.
	$\dfrac {y'x - y} {x} - 1 = 0$.
	$y' = \dfrac {x + y} {x}$.
	$\zed' = \dfrac {\dfrac {x + \zed} {x} - 1} {\dfrac {x + \zed} {x} + 1} = \dfrac {\zed} {2x + \zed}$.
	$u\br{x} = \dfrac {\zed\br{x}} {x}$.
	$\zed' = u'x + u = \dfrac {u} {2 + u}$.
	$u'x = \dfrac {u - 2u - u^2} {u + 2}$.
	$\dfrac{u + 2}{u^2 + u} u' = -\dfrac {1} {x}$,
	причём $u\br{x} = 0$ и $u\br{x} = -1$ -- решения, \vspace{1mm} из которых получаются функции $\zed\br{x} = 0$ и $\zed\br{x} = -x$.
	
	$\dfrac {u + 2} {u^2 + u} = \dfrac {2} {u} - \dfrac {1} {u + 1}$.
	$\tint \dfrac {u + 2} {u^2 + u} u' dx 
	= \tint \br{\dfrac {2} {u} - \dfrac {1} {u + 1}} du
	= 2\ln\vbr{u} - \ln\vbr{u + 1} + C$. \vspace{1mm}
	
	$2\ln\vbr{u} - \ln\vbr{u + 1} = -\ln\vbr{x} + C$.
	$u^2 x^2 = C\br{ux + x}$.
	
	Итак, общая формула изогональных траекторий имеет вид
	$\zed^2 = C\br{\zed + x}$,
	к которым необходимо добавить $\zed = -x$.
	
	\item $\br{x - 3y}^4 = Cxy^6$. \\
	$C = \dfrac {\br{x - 3y}^4} {xy^6}$.
	$0 = \dfrac {4\br{x - 3y}^3 \br{1 - 3y'} \cdot xy^6 - \br{x - 3y}^4 \br{y^6 + 6xy^5 y'}} {x^2 y^{12}}$. \\
	$4xy\br{1 - 3y'} - \br{x - 3y}\br{y + 6xy'} = 0$.
	$4xy - 12xyy' - xy + 3y^2 - 6x^2y' + 18xyy' = 0$.
	$xy + 2xyy' + y^2 - 2x^2y' = 0$.
	$y' = \dfrac {xy + y^2} {2x^2 - 2xy}$.
	$\zed' = \dfrac {-2x^2 + 3x\zed + \zed^2} {2x^2 - x\zed + \zed^2}$.
	$\zed\br{x} = x u\br{x}$. \vspace{1mm}
	$u'x + u = \linebreak 
	= \dfrac {-2 + 3u + u^2} {2 - u + u^2}$.
	$u'x = \dfrac {u^2 + 3u - 2 - 2u + u^2 - u^3} {u^2 - u + 2} 
	= -\dfrac {u^3 - 2u^2 - u + 2} {u^2 - u + 2}$. \vspace{2mm}
	Числитель разложим на множители методом группировки. \\	
	$u' x = -\dfrac {u^2\br{u - 2} - \br{u - 2}} {u^2 - u + 2} = -\dfrac {\br{u^2 - 1}\br{u - 2}}{u^2 - u + 2} = -\dfrac {\br{u - 1}\br{u + 1}\br{u - 2}} {u^2 - u + 2}$. \\
	$\dfrac {u^2 - u + 2} {\br{u - 1}\br{u + 1}\br{u - 2}} u' = - \dfrac {1} {x}$.
	Для взятия интеграла рациональную дробь в левой части разложим в сумму простейших методом вычёркивания.
	\begin{align*}
		\dfrac {u^2 - u + 2} {\br{u - 1}\br{u + 1}\br{u - 2}} = \dfrac {a} {u - 1} + \dfrac{b} {u + 1} + \dfrac{c} {u - 2}\text{;}
		&&
		a = \dfrac {1^2 - 1 + 2} {\br{1 + 1}\br{1 - 2}} = -1;
	\end{align*}
	\begin{align*}
		b = \dfrac {\br{-1}^2 - \br{-1} + 2} {\br{-1 - 1} \br{-1 - 2}} = \dfrac {2} {3};
		&& c = \dfrac {2^2 - 2 + 2} {\br{2 - 1} \br{2 + 1}} = \dfrac {4} {3};
	\end{align*}
	Итак,
	$\dfrac {u^2 - u + 2} {\br{u - 1}\br{u + 1}\br{u - 2}} = - \dfrac {1} {u - 1} + \dfrac{2} {3\br{u + 1}} + \dfrac{4} {3\br{u - 2}}$. \\
	$\br{- \dfrac {1} {u - 1} + \dfrac{2} {3\br{u + 1}} + \dfrac{4} {3\br{u - 2}}} u' = -\dfrac {1} {x}$. 
	$\dfrac {2} {3} \ln\vbr{u + 1} + \dfrac {4} {3} \ln\vbr{u - 2} - \ln\vbr{u - 1} = -\ln\vbr{x} + C$.
	$\dfrac {\br{u + 1}^2\br{u - 2}^4} {\br{u - 1}^3} = \dfrac {C} {x^3}$.
	
	Получаем изогональные траектории в параметрической форме:
	$$\tsys{& x\br{u} = C f\br{u} \\& \zed\br{u} = C u f\br{u}} \text{, где } f\br{u} = \dfrac {u - 1} {\sqrt[3]{\br{u + 1}^2\br{u - 2}^4}}.$$
	
	Из соотношения
	$\dfrac {\br{u + 1}^2\br{u - 2}^4} {\br{u - 1}^3} = \dfrac {C} {x^3}$
	также можно получить уравнения изогональных траекторий в неявной форме $\br{u + 1}^2 x^2 \cdot \br{u - 2}^4 x^4 = C \br{u - 1}^3 x^3$.
	
	$\br{\zed + x}^2 \cdot \br{\zed - 2x}^4 = C \br{\zed - x}^3$.
	
	В процессе решения также были потеряны функции $u\br{x} = -1$, $u\br{x} = 2$ и $u\br{x} = 1$. При $C = 0$ первые две попадают в общую формулу, а последняя требует записи отдельно:
	$\zed\br{x} = x$.
\end{enumerate}
