$\ddot{x} = 1 - \tpow{e}{x}$.
$\tsys{& \dot{x}\br{t} = y \\& \dot{y}\br{t} = 1 - \tpow{e}{x}}$;
$\dfrac {dy} {dx} = \dfrac {1 - \tpow{e}{x}} {y}$.
$2y \dfrac {dy} {dx} = 2 - 2\tpow{e}{x}$.
$y^2 = 2x - 2\tpow{e}{x} + C$. \\
В качестве уравнения траектории будем использовать
$y^2 + 2\br{\tpow{e}{x} - x - 1} = C$, $C > 0$.

Особой точкой будет $M\br{0; 0}$. $1 - \tpow{e}{x} = 1 - \br{1 + x + \dfrac {1}{2} x^2 + o\br{x^2}} = -x + O\br{x^2}$.
$\dfrac {dy} {dx} = \dfrac {-x + O\br{x^2}} {y}$.
$\lambda^2 + 1 = 0$.
$\lambda_{1} = -i$, $\lambda_{2} = i$.
Поскольку траектории обладают симметрией относительно оси $Ox$, $M\br{0; 0}$ -- <<центр>>.
