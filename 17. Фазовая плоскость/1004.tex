Уравнение $\ddot{x}\br{t} = 3x^2$
не меняет вид от замены независимой переменной
$t = -u$,
так что поведение решений при $t \to +\infty$ и при $t \to -\infty$ будет одинаковым.

$\tsys{& \dot{x}\br{t} = y \\& \dot{y}\br{t} = 3x^2}$;
$\dfrac {dy} {dx} = \dfrac {3x^2} {y}$.
$6x^2 = 2yy'$,
$y^2 = 2x^3 + C$.

Единственной особой точкой будет $E\br{0; 0}$.
$\dfrac {dy} {dx} = \dfrac {O\br{x^2}} {y}$.
$A = \begin{pmatrix} 0 & 1 \\ 0 & 0 \end{pmatrix}$.
$\lambda^2 = 0$. $\lambda_{1} = \lambda_{2} = 0$.
Таким образом, $E\br{0; 0}$ не относится к обычным особым точкам. \vspace{1mm}

Следовательно, всякое решение задачи Коши 
$\ddot{x}\br{t} = 3x^2$,
$x\br{t_0} = a$,
$\dot{x}\br{t_0} = b$
при $t \to +\infty$ будет стремиться к $+\infty$,
кроме, случаев $b^2 = 2a^3$, когда возможно стремление к началу координат.

Более точно, $b^2 = 2a^3$ даёт возможность решить уравнение в квадратурах; сами решения \linebreak имеют форму 
$x\br{t} = \br{C \pm \frac {t} {\sqrt{2}}}^{-2}$. Из этого видно, что решения такого вида не всегда продолжимы на $t \to +\infty$. При этом, если решение конкретной задачи Коши продолжимо, то оно стремится к нулю, иначе $x \to +\infty$ при некотором конечном $t$.

Неустойчивость нулевого решения системы \linebreak
$\tsys{& \dot{x}\br{t} = y \\& \dot{y}\br{t} = 3x^2}$
также можно легко показать, используя функцию Ляпунова $v\br{x; y} = xy$ в области
$V = \cbr{x > 0, y > 0}$, для которой
$\dfrac {dv} {dt} \bigg| = y^2 + 3x^3 > 0$ внутри области $V$.
