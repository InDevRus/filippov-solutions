# filippov-solutions

Здесь собраны мои собственные решения задач из задачника "Сборник задач по дифференциальным уравнениям" за авторством А.
Ф. Филиппова.

Компилировать их можно на движках XeTeX или LuaLaTeX с полной установкой всех пакетов в сборке (например, TeXLive). 
В том числе отрисовка на [overleaf.com](https://overleaf.com) должна быть проведена одним из этих движков.

**_Внимание!_** Некоторые решения содержат графики, которые отрисовываются при помощи инфраструктуры
"[PgfPlots](https://pgfplots.sourceforge.net/pgfplots.pdf)", из-за чего компиляция становится существенно
продолжительнее.
