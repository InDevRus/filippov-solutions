$y = C \br{x - C}^2$.
$y' = 2C\br{x - C}$.
Обозначим $\br{x - C} = \delta$ и решим систему
$\tsys{&C \delta^2 = y \\& 2C\delta = y'}$.
$C\delta = \dfrac {y'} {2}$.
$y = C \delta ^2 = C \delta \cdot C = C \dfrac {y'} {2}$.
$C = \dfrac {2y} {y'}$.
$y' = \dfrac {4y} {y'} \br{x - \dfrac {2y} {y'}}$.
$\br{y'}^3 = 4y \br{xy' - 2y}$.
