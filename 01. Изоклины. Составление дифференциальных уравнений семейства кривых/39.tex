\begin{framed}
	Для угла $\phi = \ang{90}$ имеем $\tg{\beta} = \tg\br{\alpha \pm \phi} =\tg\br{\alpha \pm \ang{90}} = -\ctg \alpha$.
\end{framed}

$x^2 = y + Cx$.
$C = \dfrac {x^2 - y} {x}$.
$0 = \dfrac {\br{2x - y'}x - \br{x^2 - y}} {x^2} = \dfrac {x^2 - xy' + y} {x^2}$.
Имеем дифференциальное уравнение: $y' = \dfrac {y + x^2} {x}$. 
$\tg\alpha = \dfrac {y + x^2} {x}$.
$\tg\beta = -\ctg\alpha = -\dfrac {x} {x^2 + y}$. Окончательно, уравнение изогональных (ортогональных) траекторий имеет вид $\zed' = -\dfrac {x} {x^2 + \zed}$.

\begin{remark}
	Уравнение $\zed' = -\dfrac {x} {x^2 + \zed}$ имеет общее решение, например, в неявном виде: $x^2 = \dfrac {1} {2} - \zed + \widetilde{C} e^{-2\zed}$, $\widetilde{C} > -\dfrac {1} {2}$.
\end{remark}
