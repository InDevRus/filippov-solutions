\begin{framed}
	Для угла $\phi = \ang{90}$ имеем $\tg{\beta} = \tg\br{\alpha \pm \phi} =\tg\br{\alpha \pm \ang{90}} = -\ctg \alpha$.
\end{framed}

$y = Cx^4$.
$C = \dfrac {y} {x^4}$.
$0 = \dfrac {x^4 y' - 4x^3 y} {x^8}$.
Итак, дифференциальное уравнение таково: $y' = \dfrac {4y} {x}$. 
$\tg\alpha = \dfrac {4y} {x}$.
$\tg\beta = -\ctg\alpha = -\dfrac {x} {4y}$. Окончательно, уравнение изогональных (ортогональных) траекторий имеет вид $\zed' = -\dfrac {x} {4\zed}$.
$4\zed\zed' = -x$.

\begin{remark}
	Уравнение $4\zed\zed' = -x$ имеет общее решение $4\zed^2 + x^2 = \widetilde{C}$.
\end{remark}
