Требуемые окружности имеют центры с координатами $\br{a; 2a}$, а потому их уравнения имеют вид $\br{x - a}^2 + \br{y - 2a}^2 = 1$.

$2\br{x - a} + 2\br{y - 2a}y' = 0$.
$a = \dfrac {x + yy'} {2y' + 1}$.
$\br{x - \dfrac {x + yy'} {2y' + 1}}^2 + \br{y - 2 \cdot \dfrac {x + yy'} {2y' + 1}}^2 = 1$. \\
$\br{\dfrac {2xy' + x - x - yy'} {2y' + 1}}^2 + \br{\dfrac {2yy' + y - 2x - 2yy'} {2y' + 1}}^2 = 1$. \\
$\br{2xy' - yy'}^2 + \br{y - 2x}^2 = \br{2y' + 1}^2$.
$\br{y - 2x}^2 \br{\br{y'}^2 + 1} = \br{2y' + 1}^2$.
