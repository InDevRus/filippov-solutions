\begin{framed}
	Для угла $\phi = \ang{60}$ имеем $\tg{\beta} =
	\tg\br{\alpha \pm \phi} =
	\tg\br{\alpha \pm \ang{60}} =
	\dfrac {\tg\alpha \pm \sqrt{3}} {1 \mp \sqrt{3}\tg\alpha}$.
\end{framed}

$y = kx$.
$y' = \dfrac {y} {x}$.
$\tg \alpha = \dfrac {y} {x}$.

В случае со знаком <<плюс>>
$\tg \beta = \dfrac {\tg\alpha + \sqrt{3}} {1 - \sqrt{3}\tg\alpha} 
= \dfrac {y + \sqrt{3}x} {-\sqrt{3}y + x}
= g\br{x; y\br{x}}$,
а в случае со знаком <<минус>> 
$\tg \beta = \dfrac {\tg\alpha - \sqrt{3}} {1 + \sqrt{3}\tg\alpha} 
= \dfrac {y - \sqrt{3}x} {\sqrt{3}y + x}
= g\br{x; y\br{x}}$.
Таким образом, имеем два уравнения изогональных траекторий вида $\zed'\br{x} = f\br{x; \zed\br{x}}$ и $\zed'\br{x} = g\br{x; \zed\br{x}}$, то есть $\zed' = \dfrac {\zed + \sqrt{3}x} {-\sqrt{3}\zed + x}$ и $\zed' = \dfrac {\zed - \sqrt{3}x} {\sqrt{3}\zed + x}$.

\begin{remark}
	Ниже приведены решения соответствующих уравнений в параметрической форме $\tsys{& x\br{t} = C f\br{t} \\& \zed\br{t} = C t f\br{t}}$ и в полярной форме $\tsys{& x\br{\theta} = C r\br{\theta} \cos\theta \\& \zed\br{\theta} = C r\br{\theta} \sin\theta}$.
