\begin{framed}
	Для угла $\phi = \ang{60}$ имеем $\tg{\beta} =
	\tg\br{\alpha \pm \phi} =
	\tg\br{\alpha \pm \ang{60}} =
	\dfrac {\tg\alpha \pm \sqrt{3}} {1 \mp \sqrt{3}\tg\alpha}$.
\end{framed}

$3x^2 + y^2 = C$.
$6x + 2yy' = 0$.
$y' = \tg\alpha = - \dfrac {3x} {y}$.

Имеем
$\tg \beta 
= \dfrac {\sqrt{3} \tg\alpha \pm 1} {\sqrt{3} \mp \tg\alpha}
= \dfrac {3\sqrt{3}x \pm y} {\sqrt{3}x \mp 3y} $.
Обозначим
$\dfrac {3\sqrt{3}x + y} {\sqrt{3}y - 3x} = g\br{x; y\br{x}}$ и 
$\dfrac {3\sqrt{3}x - y} {\sqrt{3}y + 3x} 
\linebreak = h\br{x; y\br{x}}$,
и получается уравнения изогональных траекторий вида $\zed'\br{x} = g\br{x; \zed\br{x}}$ и $\zed'\br{x} = h\br{x; \zed\br{x}}$,
то есть $\zed' = \dfrac {3\sqrt{3}x + \zed} {3x - \sqrt{3}\zed}$
и $\zed' = \dfrac {\zed - 3\sqrt{3}x} {\sqrt{3}\zed + 3x} $.
