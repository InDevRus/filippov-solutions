$y' = xy^2 + 2xy$.
$y' = x\br{y^2 + 2y}$.
–§—É–Ω–∫—Ü–∏–∏ $y\br{x} = 0$ –∏ $y\br{x} = -2$ —è–≤–ª—è—é—Ç—Å—è —Ä–µ—à–µ–Ω–∏—è–º–∏.
$\dfrac {y'} {y\br{y + 2}} = x$.
–ò–º–µ–µ–º $\dfrac {1} {y\br{y + 2}} = \dfrac {1} {2} \br{ \dfrac {1} {y} - \dfrac {1} {y + 2}}$, —Ç–∞–∫ —á—Ç–æ \vspace{1mm}
$\tint \dfrac {y'} {y\br{y + 2}} dx = \dfrac {1} {2} \tint \dfrac {dy} {y} - \linebreak - \dfrac {1} {2} \tint \dfrac {dy} {y + 2} = \dfrac {1} {2}\br{\ln\vbr{y} - \ln\vbr{y + 2}} + C$.

$\ln\vbr{y} - \ln\vbr{y + 2} = x^2 + C$.
$\dfrac {y} {y + 2} = C\tpow{e}{x^2}$. –ü—Ä–∏ —Ç–∞–∫–æ–π —Ñ–æ—Ä–º–µ –æ–±—â–µ–≥–æ —Ä–µ—à–µ–Ω–∏—è $y\br{x} = 0$ –¥–æ—Å—Ç–∏–≥–∞–µ—Ç—Å—è –ø—Ä–∏ $C = 0$, –∞ —Ä–µ—à–µ–Ω–∏–µ $y\br{x} = -2$ –±—É–¥–µ—Ç –¥–æ–ø–æ–ª–Ω–∏—Ç–µ–ª—å–Ω—ã–º.
$1 - \dfrac {2} {y + 2} = C\tpow{e}{x^2}$.
$y\br{x} = \dfrac {2} {1 + C\tpow{e}{\elevate{x^2}}} - 2$.

–û–∫–æ–Ω—á–∞—Ç–µ–ª—å–Ω–æ, –æ–±—â–µ–µ —Ä–µ—à–µ–Ω–∏–µ —Ä–∞–≤–Ω–æ
$y\br{x} = \dfrac {2} {1 + C\tpow{e}{\elevate{x^2}}} - 2$,
–∏ –¥–æ–ø–æ–ª–Ω–∏—Ç–µ–ª—å–Ω–æ $y\br{x} = -2$.
