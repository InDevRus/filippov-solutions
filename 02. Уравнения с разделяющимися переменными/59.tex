$\tpow{e}{-s}\br{1 + \dfrac {ds} {dt}} = 1$, где $s = s\br{t}$.
$\dfrac {ds} {dt} = \tpow{e}{s} - 1$.
$\dfrac {s'\br{t}} {\tpow{e}{s} - 1} = 1$,
а функция $s\br{t} = 0$ относится к решениям.
$\tint \dfrac {s'\br{t}} {\tpow{e}{s} - 1} dt
= \tint \dfrac {ds} {\tpow{e}{s} - 1} = 
= \begin{bmatrix}
	s = \ln u \\
	ds = \dfrac {du} {u}
\end{bmatrix} 
= \tint \dfrac {du} {u\br{u - 1}}
= \tint \br{\dfrac {1} {u - 1} - \dfrac {1} {u}} du = \linebreak
= \ln\vbr{u - 1} - \ln\vbr{u} + C
= \ln\vbr{1 - \tpow {e}{-s}} + C$.

Итак, взятие интеграла обеих частей уравнения 
$\dfrac {s'\br{t}} {\tpow{e}{s} - 1} = 1$.
даёт $\ln\vbr{\tpow {e}{-s} - 1} = t + C$.
$\tpow{e}{-s} = 1 + C\tpow{e}{t}$.
$s\br{t} = -\ln\vbr{1 + C\tpow{e}{t}}$,
а функция $s\br{t} = 0$ входит в такое общее решение при $C = 0$.
