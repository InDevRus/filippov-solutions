$x \dfrac {dx} {dt} + t = 1$.
$x \dfrac {dx} {dt} = 1 - t$.
$\tint x \dfrac {dx} {dt} dt
= \tint x d\br{x\br{t}}
= \dfrac {1} {2} x^2\br{t} + C$.

$\dfrac {1} {2} x^2 = t - \dfrac {t^2} {2} + C$.
$x^2 + t^2 - 2t = C$.\vspace{1mm}

$x^2 + \br{t - 1}^2 = C$, где $C > 0$.
