$x y' = y^2 - y$.
$\dfrac {y'} {y^2 - y} = \dfrac {1} {x}$, причём $y = 0$ и $y = 1$ также являются решениями.

$\tint \dfrac {y'(x)} {y^2(x) - y(x)} dx = 
\tint \dfrac {1} {y^2 - y} dy = 
\tint \br{ \dfrac {1 - y} {y\br{y - 1}} + \dfrac {y} {y\br{y - 1}}} dy = 
\tint \br{ \dfrac {1} {y - 1} - \dfrac {1} {y}} dy \linebreak =
\ln\vbr{y - 1} - \ln\vbr{y} + C = \ln\vbr{1 - \dfrac {1}{y}} + C$.

$\ln\vbr{1 - \dfrac {1}{y}} = \ln\vbr{x} + C$.
$1 - \dfrac {1} {y} = Cx$.
$y = \dfrac {1} {1 + Cx}$, а решение $y = 1$ получается при $C = 0$.

Итак, общее решение имеет вид $y = \dfrac {1} {1 + Cx}$, кроме которого имеется дополнительное $y = 0$.

Решение задачи с начальным условием $y\br{1} = 0,5$ таково, что $0,5 = \dfrac {1} {1 + C}$, $C = 1$. Значит, $y\br{x} = \dfrac {1} {1 + x}$ в области $x > -1$.
