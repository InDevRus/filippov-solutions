$\sqrt{y^2 + 1} dx = xy\ dy$. 
Кривая $x = 0$ также является решением.
$\dfrac {1} {x} dx = \dfrac {y} {\sqrt{y^2 + 1}} dy$.

$\tint \dfrac {1} {x} dx = \ln\vbr{x} + C$;
$\tint \dfrac {y} {\sqrt{y^2 + 1}} dy = \dfrac {1} {2} \tint \tpow{\br{y^2 + 1}} {-\frac {1} {2}} d\br{y^2} = \sqrt{y^2 + 1} + C$.

$\ln \vbr{x} + C = \sqrt{y^2 + 1}$. $y^2 = \br{\ln \vbr{x} + C}^2 - 1$ при условии $\vbr{x} \ge {e}^{-C}$.

Получаем общее решение в явном виде $y = \pm \sqrt{\br{\ln \vbr{x} + C}^2 - 1}$, $\vbr{x} \ge {e}^{-C}$ и дополнительное $x = 0$.

    \begin{tikzpicture}
        \pgfplotsset{
            Graph/.style = {
                    samples = 100,
                },
        }
        \begin{axis}[
            width = 14cm,

            ymin = -3.2,
            ymax = 3.2,
            xmin = -8.2,
            xmax = 8.2, 
            xtick = {-10, ..., 10},
            ytick = {-10, ..., 10},
            minor tick num = 3,
            declare function = {
                f(\C, \x) = \C * exp(sqrt(x^2 + 1));
            }]
            
            \addplot[domain = -4 : 4, thick] ({f(0, x)}, {x});
            \foreach \s in {-1, 1} {
                \foreach \C in {0.1, 0.25, 0.5, 1, 2} {
                    \addplot[Graph, domain = -3 : 3] ({f(\s * \C, x)}, {x});
                }
            }
                        
        \end{axis}
    \end{tikzpicture}
    \centering
\end{figure}
